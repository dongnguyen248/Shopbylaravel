## How to install

-   **Clone Reposesitory**
-   **composer install**
-   **php artisan migrate** -**php artisan db:seed --class=DummyUsersSeeder**
-   **php artisan passport:keys**
-   **npm install**
-   **php artisan serve**
-   **enjoy**
